# Role Application UI

This is a multi-step Role Application form built for the **La Tanda** platform.  
It includes role selection, user information input, document upload with preview, progress tracking, and status updates with mock API integration.

## Features
- Multi-step form with progress bar
- Role selection (dropdown)
- User information input (name, email)
- File upload with preview (images or PDF)
- Status tracking page (Pending / Approved / Rejected)
- Mock API integration with async behavior
- Responsive and mobile-first design
- Dark theme with cyan (#00FFFF) accents

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript

## Usage
1. Open `index.html` in a web browser.
2. Complete all steps:
   - Select a role
   - Fill in your name and email
   - Upload a document
3. Click Submit to see the status page with mock API response.
